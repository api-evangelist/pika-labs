# Pika Labs (pika-labs)

Pika Labs is an AI video generation platform offering text-to-video, image-to-video, and video editing capabilities. As of May 2026 Pika does not publish a first-party REST API on its main domain; production API access is provided through partner aggregators (notably fal.ai).
